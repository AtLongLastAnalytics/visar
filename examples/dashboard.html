About this dashboard

VISaR (Vulnerability Identification, Scanning & Reporting) is a free, open-source supply-chain vulnerability scanner built for data engineers and data-platform teams evaluating open-source software under regulated and audit-grade requirements — defence, aviation, healthcare, and other contexts where you need to evidence what was scanned, when, and what changed between scans. It works just as well for individual developers vetting a single project.

The dashboard is a single self-contained HTML file. No network requests, no trackers, no external fonts — safe to open in air-gapped review environments. Every inline script and style is SHA-256 pinned in the CSP.

How to use it

  • Pick a repo and scan date from the two dropdowns at the top of the controls.
  • Filter by severity by clicking any of the six cards (Critical, High, Moderate, Low, Other, Total). Click Total to clear the filter.
  • Search by vulnerability ID (CVE / GHSA / PYSEC) or by any text inside the details field.
  • Diff against the previous scan — when a prior scan of the same repo exists, the comparison bar shows what changed. Use Changes only to hide unchanged rows, or Show resolved to bring back rows that have dropped out of the latest scan.
  • Copy a vulnerability ID with the small clipboard button next to it — useful when pasting into a ticket.
  • Download CSV exports the currently-visible rows (respecting filter, search, and the active diff) for sharing with developers or attaching to an audit packet.
